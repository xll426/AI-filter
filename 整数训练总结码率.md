# 3. 结论

1. **相对于原始编码（org）**：
   - ref 与 w10b13 在两个分辨率下均实现：
     - **PSNR 提升**
     - **码率下降**
   - ref 在两组数据中均取得更高的 PSNR 提升。
   - w10b13 在 4K 场景中实现了更高的码率压缩效率。

2. **ref 与 w10b13 对比**：
   - 在 3840×2160 场景：
     - w10b13 在 PSNR 基本保持的前提下，实现 **额外 5.2% 码率下降**
     - 但 Flick 指标显著上升（+10.4%）
   - 在 2560×1440 场景：
     - w10b13 出现 **PSNR下降（-0.43 dB）且码率上升（+2.6%）**
     - Flick 略有改善
